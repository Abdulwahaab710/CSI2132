
# Jobify
Our project for Database I (CSI2132)
## To run the web app
### requirements
- Ruby == 2.4.0
- Postgresql

Installation
---
- Clone the repo
```bash
git clone git@github.com:Abdulwahaab710/CSI2132.git
```
- change directory
```bash
cd CSI2132/jobify
```
- Set the env to production
```bash
RAILS_ENV=production
```
- Run bundle installer
```bash
bundle install
```
- Database migration
```bash
rake db:migrate
```
- Run rails
```
rails s
```
