
# Jobify
Our project for Database I (CSI2132)
## To run the web app
### requirements
- Ruby == 2.4.0
- Postgresql

Installation
---
```bash
# Clone the repo

git clone git@github.com:Abdulwahaab710/CSI2132.git

# change directory

cd CSI2132/jobify

# Set the env to production

RAILS_ENV=production

# Run bundle installer

bundle install

# Database migration

rake db:migrate

# Run rails

rails s
```
